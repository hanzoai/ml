# ML MNIST Tutorial

## Introduction

This tutorial provides an introduction to ML by implementing and training a neural network for MNIST digit classification from scratch. 

Throughout this tutorial, you will learn the basics of:

- Tensor operations and model construction
- Creating and implementing neural network layers
- Parameter initialization
- Training loop implementation
- Saving and loading trained models

## Getting Started

Before proceeding, please ensure that you have properly installed ML by following the instructions in the [Installation](../installation.md) guide.
